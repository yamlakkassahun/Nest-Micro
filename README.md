## Description

This project is a simple micro services application using nestjs and rabbitmq as a message broker and postgres daatabase to store data and docker to compose the project.

## Running the app

```bash
# compose the project
$ docker compose up

```

## Stop the app

```bash
# compose the project
$ docker compose down

```
